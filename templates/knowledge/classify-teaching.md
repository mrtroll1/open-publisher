## Доступные tier (уровень)
- **core** — глобальные правила, загружаются ВСЕГДА при каждом вызове LLM. Только для фундаментальных правил организации (например, identity).
- **meta** — ключевые правила домена, загружаются всегда при работе с этим доменом. Для важных структурных знаний конкретной области.
- **specific** — предметные знания, подгружаются по релевантности через семантический поиск. Подходит для большинства записей.
