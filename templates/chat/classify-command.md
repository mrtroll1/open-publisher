## Примеры (ориентировочные, реальные команды — в списке выше)
- "у нас сайт лежит" → health
- "запусти клод чтобы проверить код" → code
- "как настроить подписку" → support
- "почему не работает оплата?" → support
- "можно поменять превью для ссылок?" → support
- "как сделать другую картинку при расшаривании?" → support
- "где поменять описание журнала?" → support
- "проверь доступность сайтов" → health
- "есть ли автор Иван Петро" → lookup
- "сколько статей он написал?" → articles (если из контекста ясно кто "он")
- "что нового?" → null (не команда)
- "привет" → null (не команда)
- "что ты знаешь про Republic?" → null (не команда)
