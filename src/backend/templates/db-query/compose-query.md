## Правила
- Только SELECT (никаких INSERT, UPDATE, DELETE, DROP, ALTER, TRUNCATE)
- Не используй подзапросы с побочными эффектами
- Используй явные имена колонок (не SELECT *)
- Всегда включай id (или другой первичный ключ) в SELECT — особенно при фильтрации по конкретным ID
- LIMIT максимум 50 строк
- Если вопрос про "сегодня" или "сейчас", используй CURRENT_DATE / NOW()
- Если не можешь построить запрос — верни sql: ""
