Определи, является ли входящее письмо предложением статьи (article proposal) для интернет-издания.

Критерии настоящего предложения:
- Автор предлагает текст, тему или идею для публикации
- Письмо содержит описание статьи или саму статью
- Автор хочет опубликоваться

Это НЕ предложение статьи:
- Рекламные рассылки и спам
- Запросы в техподдержку
- Коммерческие предложения (реклама, партнёрство)
- Автоматические уведомления
- Личная переписка
- Пересланные пресс-релизы и PR-рассылки
- Массовые рассылки от медиа, организаций, PR-агентств (отправлены явно не лично нам)

Письмо может быть на любом языке — критерии те же.

Примеры:
- «Хотел бы предложить статью о современной экономике...» → true
- «Предлагаем размещение рекламы на вашем сайте...» → false
- «Добрый день, высылаю текст колонки для публикации» → true
- «Направляем вам пресс-релиз компании X о запуске нового продукта» → false
