Извлеки данные пользователя из текста.

## Текст
{{TEXT}}

## Задача

Определи из текста:
- **telegram_id** — числовой Telegram ID (если есть, может быть в формате [telegram_id=123456789])
- **name** — имя/фамилия человека
- **role** — роль: user, editor, admin (по умолчанию user)
- **email** — email (если есть)

Верни JSON: {"telegram_id": 123456789, "name": "Имя Фамилия", "role": "editor", "email": null}
Если поле не удалось определить — верни null для него.
