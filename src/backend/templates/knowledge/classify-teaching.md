## Доступные tier (уровень)
- **core** — глобальные правила, загружаются ВСЕГДА при каждом вызове LLM. Только для фундаментальных правил организации (например, identity).
- **meta** — ключевые правила домена, загружаются всегда при работе с этим доменом. Для важных структурных знаний конкретной области.
- **specific** — предметные знания, подгружаются по релевантности через семантический поиск. Подходит для большинства записей.

## Доступные visibility (кто видит)
- **public** — все пользователи. Общие знания, факты, правила.
- **environment** — только пользователи того же окружения (чата) + администраторы. Рабочие решения, внутренние обсуждения.
- **role:editor** — редакторы и администраторы. Редакционные процессы, внутренняя кухня.
- **role:admin** — только администраторы. Финансы, зарплаты, персональные данные, стратегия.
- **user** — только конкретный пользователь + администраторы. Персональные данные контрагента.
